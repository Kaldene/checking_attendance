## Описание проекта

Веб-приложение для **автоматической фиксации посещаемости** с помощью **распознавания лиц**.  
Использует:
- **Streamlit** — веб-интерфейс
- **OpenCV + DeepFace** — камера и распознавание
- **SQLite** — база данных
- **Python 3.12**

---

## Возможности

- Регистрация студентов (имя, фамилия, группа, фото)
- Распознавание лиц через веб-камеру (8 секунд)
- Сохранение посещаемости в базу
- Защита паролем (`admin123`)
